**Dependencies**
- Node.js 18.x.x LTS
- Hosting, (VPS, Dedicated Server, Bot hosting) We recommend at least 512mb of RAM and at least 1 core
- A text editor, We recommend Visual Studio Code
- A stable Internet Connection
